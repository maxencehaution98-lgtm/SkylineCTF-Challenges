# Ping Quest

Catégorie  : Web
Difficulté : Basique

## Description :

Un administrateur système a mis en place un outil interne pour vérifier la latence réseau des serveurs. Cependant, il semble qu'il ait oublié de sécuriser correctement les entrées utilisateur... Sauras-tu exploiter cette faille pour récupérer le flag caché ?
